# AI Resume Parser

### Transformer-Powered Resume Intelligence Engine

An AI-powered Resume Parser built using Transformer models, semantic embeddings, and interactive analytics to evaluate resumes with intelligent scoring and job-description matching.

## Features

* Resume parsing using BERT-based NER
* Semantic skill matching with Sentence Transformers
* Job Description similarity scoring
* 7-Dimension Resume Scoring:

  * Confidence
  * Completeness
  * ATS Readiness
  * Impact
  * Skills
  * Experience
  * Presentation
* Career intelligence:

  * Role classification
  * Industry inference
  * Seniority detection
* Achievement extraction and action verb analysis
* Career gap and progression detection
* Interactive Plotly visualizations:

  * Radar chart
  * Skill distribution
  * Timeline analysis
  * Comparative score bars
* Modern glassmorphism UI using Streamlit/Gradio

---

## Tech Stack

| Layer             | Technology                              |
| ----------------- | --------------------------------------- |
| NER               | `dslim/bert-base-NER`                   |
| Semantic Matching | `all-MiniLM-L6-v2`                      |
| Visualization     | `plotly`                                |
| UI                | `streamlit` / `gradio`                  |
| NLP               | `transformers`, `sentence-transformers` |
